# Ukraine Primary Enrollment and GDP Dataset

This repository contains the data and files created for a data wrangling assignment analyzing the relationship between primary school enrollment and GDP per capita in Ukraine from 2001 to 2020.

## Files Included

- **PrimaryEnrollment_GDP.csv**  
  Cleaned CSV dataset containing yearly primary enrollment counts and GDP per capita values for Ukraine (2001â€“2020).

- **ukraine_edu_gdp.db**  
  SQLite database containing the same dataset stored in a relational table.

- **attributes.json**  
  JSON file describing the dataset schema, including column names, data types, and the meaning of each column.

## Description

The dataset was created by combining primary education enrollment data with GDP per capita data and restructuring it into a clean, analysis-ready format. The CSV and SQLite database contain identical data, while the JSON file provides metadata describing the dataset structure.
